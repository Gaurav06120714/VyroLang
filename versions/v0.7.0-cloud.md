# v0.7.0 — VyroCloud + Sandbox

**Theme:** Safe cloud execution at the core.

## Scope
- Docker sandbox runner: CPU/RAM/time/FS limits, default-deny net, cap-drop, seccomp.
- Orchestrator + Redis job queue; stream results to IDE.
- Workspaces (persist files + metadata).

## Deliverables
- Cloud run path with hard isolation.
- Sandbox escape test suite (must fail to escape).

## Exit Criteria
- Fork bombs/OOM/infinite loops contained; host unaffected; escape tests pass.
