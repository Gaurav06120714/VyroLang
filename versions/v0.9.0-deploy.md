# v0.9.0 — VyroDeploy + Hardening

**Theme:** Ship apps to live URLs, securely.

## Scope
- Push-to-deploy: build → image → Nginx route → public URL; rollbacks; secret store.
- VPS hardening: SSH key-only, no root/password, Fail2Ban, firewall, Cloudflare, rate limits, audit logs.
- CI/CD: build/test/lint/scan → auto-deploy with smoke test + rollback.
- Full security pass + pen-test checklist.

## Deliverables
- A reference app deployed to a public URL.
- Hardened, monitored VPS stack.

## Exit Criteria
- Deploy + rollback work; hardening checklist complete; security tests pass.
