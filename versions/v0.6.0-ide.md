# v0.6.0 — VyroIDE Alpha

**Theme:** Write and run VyroLang in the browser.

## Scope
- Next.js + Monaco shell; `.vy` highlighting.
- Compiler API + inline diagnostics.
- Integrated terminal + run loop (WebSocket).
- File explorer + workspace persistence.
- LSP (completions/hover); debugger (breakpoints/step).
- Reference apps built: Todo, Calculator, Tic-Tac-Toe.

## Deliverables
- Browser write→compile→run loop.
- Three working reference apps.

## Exit Criteria
- A new user writes and runs a program in the IDE without local setup.
