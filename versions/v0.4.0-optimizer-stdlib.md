- Language: classes, async/await, error handling, modules.
- Optimizer passes: constant folding, dead-code elimination, inlining, loop optimization.

- Standard library v1: math, string, array, json, fs, http.

## Deliverables
- Optimized bytecode (passes toggleable for tests).
- Stdlib documented + tested.

## Exit Criteria
- Async + classes work; optimizer measurably reduces instruction counts.
