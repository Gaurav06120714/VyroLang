# v0.5.0 — Package Manager + OS Layer

**Theme:** Dependencies and a safe system surface.

## Scope
- `vpm`: init/install/update/remove/publish/search.
- SemVer resolver + `vyro.lock`.
- Registry service (Postgres + object storage, auth).
- VyroOS capability layer: FS/Process/Memory/Net + capability gate.

## Deliverables
- Publish a package; depend on it reproducibly.
- VyroOS APIs callable from VyroLang behind capabilities.

## Exit Criteria
- Round-trip publish→install works; capability gate denies by default.
