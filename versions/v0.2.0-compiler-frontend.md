# v0.2.0 — Compiler Frontend

**Theme:** Source becomes a validated AST.

## Scope
- Lexer with source spans.
- Recursive-descent + Pratt parser → AST.
- Semantic analysis: symbol table, scopes, basic type inference/checking.
- Diagnostics framework (codes, spans, hints).

## Deliverables
- `vyroc check main.vy` reports precise errors.
- Golden tests for AST + negative tests for diagnostics.

## Exit Criteria
- Valid programs parse + type-check; invalid ones produce correct errors.
