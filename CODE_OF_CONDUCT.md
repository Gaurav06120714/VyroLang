# Code of Conduct

## Purpose

The Vyro Ecosystem aims to be a welcoming, professional, and respectful technical space.

## Standards

Expected behavior: be respectful, assume good intent, give and accept constructive feedback, focus on what is best for the project.

Unacceptable behavior: harassment, personal attacks, discriminatory language, publishing others' private information, or any conduct that would be inappropriate in a professional setting.

## Scope

Applies to all project spaces (repositories, issues, PRs, discussions) and to public spaces when representing the project.

## Enforcement

This project is currently maintained solely by Gaurav. Concerns may be raised privately to the maintainer via GitHub. The maintainer may remove, edit, or reject contributions and comments that violate this code.

This Code of Conduct is adapted from the spirit of the Contributor Covenant.
